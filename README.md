# SafeApp — Telegram-style E2EE Messenger

## Quick start
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Features
- Telegram-style UI (dark/light)
- End-to-end encryption (RSA-OAEP + AES-GCM + PBKDF2)
- Real-time messaging via WebSocket
- Voice messages (record & send)
- File & image sharing (encrypted)
- Message reactions, reply/quote, delete
- Typing indicators
- Online/offline + last seen
- Unread badges
- Pinned contacts
- Mute conversations
- Message search
- Push notifications
- Sound notifications
- Profile photos
- No logout on refresh (session persists)
- Light/dark theme
